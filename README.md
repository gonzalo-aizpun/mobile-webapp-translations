mobile-webapp-translations
==========================

# How to install it

```shell
sudo npm install -unsafe-perm
```

# Run

```shell
node index
#or
npm start
```

# Usage

  1. Open your browser 
  2. Enter url [`localhost:4000`](http://localhost:4000)
  3. Enter keyword in the focus input
  4. And press `enter` or click in button `Search`
